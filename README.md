# generate-golang
That is simple tool to generate Golang content file structor for linux

## How to install:

* ### Step 1:

<pre>
    git clone git@github.com:danhhuynh25029/generate-golang.git
</pre>

* ### Step 2:
<pre>
    chmod +x setup.sh && ./setup.sh
</pre>

### How to use:
<pre>ggo filename</pre>

### Example:

<pre>ggo main.go</pre>
